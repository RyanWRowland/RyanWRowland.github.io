Appointment Tracker

WGU Software Development Capstone project. JavaFX desktop application designed to allow a user manage patient and therapist information and appointments between patients and therapists. Uses Java, JavaFX, and a MySQL server hosted by DigitalOcean. Secured by login, provides helpful reports and schedules, and allows user to search records by patient name.
